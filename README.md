## Featured Projects

### UNC Baseball Attendance Prediction
Collaborative MADS capstone project with Adam Cooke and Arvind Madan for UNC Athletics. Built an end-to-end attendance forecasting framework and Streamlit decision-support application for UNC baseball home games using historical attendance, opponent context, scheduling features, weather, academic calendar timing, Google Trends, promotions, and team performance data.

**My role:** Built the performance-data scraper for UNC and opponent game-level metrics, developed the Streamlit application used to present preseason and midseason forecasts interactively, contributed to project documentation, and continuously refined the scraper and app as project needs evolved.

**Partner contributions:** Adam Cooke led the GoHeels.com attendance scraper, project infrastructure, feature-tier setup, supporting datasets, GitHub organization, and stakeholder coordination. Arvind Madan led data cleaning, feature engineering, master dataset integration, backward feature elimination, weighted cross-validation, preseason workflow adaptation, and additional weather- and attendance-related feature engineering.

### CourtSpace 3D: Interactive NBA Shot Visualization App
Collaborative project with Arvind Madan. Built an interactive Streamlit app that visualizes NBA player shot attempts as 3D trajectories on a realistic half court. The app uses NBA shot chart data, Plotly 3D rendering, contextual filters, and a league-relative hot/cold zone heatmap to help users explore player shooting patterns across seasons, opponents, shot types, venues, and game periods.

**My role:** Created the first working prototype using a static LeBron James dataset, implemented the initial Streamlit version of the app, built the core visualization pipeline, added hover text metadata, quarter filtering, the full heatmap mode, a user-controlled heatmap scaling slider, and designed the caching system used to handle NBA API requests efficiently.

**Partner contributions:** Arvind Madan expanded the app with advanced filters and user interface improvements, including the max-shots slider, makes/misses filter, home/away filter, shot distance filter, shot type filter, opponent filter, improved hover text, automatically updating player and team images, and multi-season query functionality.

### Pitch Forecaster: MLB Pitch Type Prediction
Collaborative machine learning project with Arvind Madan and Elliott Walker focused on predicting the next MLB pitch type from pitch-level baseball data. The project used public Statcast/Baseball Savant data, game-context variables, pitch sequencing features, pitcher and batter tendencies, and multiclass classification models to explore how historical pitch patterns can be used to forecast future pitch selection.

**Project work:** Developed a machine learning pipeline that cleaned and transformed pitch-level data, engineered lagged sequence features, enriched observations with pitcher and batter arsenal tendencies, and evaluated multiple modeling approaches including multinomial logistic regression, random forest, neural network, and ensemble models. The final ensemble model balanced overall accuracy with stronger class-wise performance across imbalanced pitch types.

### The Three-Point Effect: Analyzing the Relationship Between NBA Three-Point Shooting and Winning
This project analyzes historical NBA team-level data to examine the relationship between three-point shooting and team success. Using year-over-year team statistics, it evaluates how three-point percentage, makes, and attempts relate to win percentage across NBA seasons.
